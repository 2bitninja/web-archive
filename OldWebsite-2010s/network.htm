Command Description OS
Remote Connections
ssh user@hostname
or
ssh hostname

-X or -Y
-C

Path: /usr/bin/		 SSH stands for Secure shell. SSH is used for securely logging into remote machines and executing commands on them. -X or -Y lets you locally display X11 programs from a remote server. -X is better to use then -Y, only use -Y if -X doesn't work.
-C is used to compress the data

The ssh configuration file.
/etc/ssh/sshd_config 		UNIX
ssh-keygen
-t rsa/dsa
-f Path		 The ssh-keygen command generates the public and private key pair, used when setting up a key exchange.
-t tells ssh-keygen which encyription standard to use. Use -f to tell the command where to place the keys.

Links to more info on setting up key exchange.
http://www.g-loaded.eu/2005/11/10/ssh-with-keys		 UNIX
ssh-copy-id The ssh-copy-id command copies your SSH public key to a remote server. This can be use instead of SCP
ssh-copy-id -i $HOME/.ssh/id_rsa.pub user@server1
ref: http://www.cyberciti.biz/faq/how-to-set-up-ssh-keys-on-linux-unix/		 UNIX
ssh-agent shell The ssh-agent command starts your ssh agent in a shell. You need to have this running if you want to log into a remote server without using your password. UNIX
ssh-add The ssh-add command adds the ssh key to your running ssh-agent. UNIX
scp Lfile username @hostname: Rdir
-r

Path: /usr/bin/ 		Securely transfer files between two computers
-r To copy a directory

Example:
scp myfile mylogin@myserver:/home
scp mylogin@myserver:/home myfile 		Links to more info.

-kill :# 		VNC is a free 3rd party program that allows a remote login, similar to Remote Desktop in Windows. Typing vncserver will start a VNC session. Adding -kill with the session number at the end of the command will terminate the session.

plumb

Path: /sbin/ or /usr/sbin/		 Used to check the computers network configuration. Similar to ipconfig in windows.
Example of bring the NIC card/interface down:
ifconfig eth1 down

The CMD plumb sets the device to a physical interface name. 		UNIX
ipconfig
/all
/renew
/release
 Displays network setting, like a computers IP address.
/all gives more details.
/renew gets new network settings from the DNS server.
/release releases the computers network settings		 Windows
